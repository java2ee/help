Использование справочной системы

Для просмотра разделов документации используйте иконку Содержании см. фрейм () слева. Кликните по выбранному разделу для его отображения в правом фрейме. Используйте кнопки навигации Назад и Вперед для перемещения по ранее просмотренным разделам.

Поиск

Чтобы быстро наити разделы документации на требуемую тему, введите в поле Поиск одно или несколько интересующих Вас слов и нажмите Выполнить, после чего в левом фрейме отобразится результат поиска. Используйте иконку Результат поиска () для отображения полученного результата поиска. Вы можете сократить объем вашего поиска, выбирая только те разделы документации, которые Вас интересуют. Нажмите ссылку Избранное и определите новый избранный список разделов или воспользуйтесь ранее созданными списками.

Синхронизация

При работе с документацией справочной системы, у Вас может произойти рассогласования содержимого левого и правого фреймов. Нажмите кнопку Показать содержание () для отображения открытой темы в структуре Содержание левого фрейма. Кнопка Обновить/показать текущий раздел () откроет содержимое текущего раздела в правом фрейме.

Избранные разделы

Переключаться между полным списком разделов документации и избранными списками можно в окне Выбрать избранный список. Для возврата к полному списку разделов в Содержании нажмите Показать все. Изменение избранного списка затрагивает Содержание и результаты поиска.

Алфавитный указатель

Для отображения отсортированного по алфавиту списка ключевых слов, кликните Алфавитный указатель (). Следует иметь ввиду, что не все разделы документации содержат алфавитные списки ключевых слов.
