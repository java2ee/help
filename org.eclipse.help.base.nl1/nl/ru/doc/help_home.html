Использование справочной системы

Просмотрите список тем во фрейме Содержания () слева. Кликните название темы для просмотра её содержания в правом фреме. Используйте кнопки Вперед и Назад для просмотра содержания тем в пределах истории ранее просматриваемых страниц.

Поиск

Чтобы быстро найти разделы документации на требуемую тему введите строку поиска в поле Найти. Используйте закладку Результаты поиска () внизу левого фрейма для просмотра найденных разделов. Для ументшения области поиска кликните на Область поиска и выберете только те разделы, которые Вас интересуют.

Синхронизация

Кликните кнопку Показать в оглавлении () для отображения открытой темы в дереве навигации левого фрейма. Кнопка Обновить/показать текущий раздел () откроет содержимое текущего раздела в правом фрейме.

Индекс ключевых слов

Чтобы показать в алфавитном порядке сортированный индекс ключевых слов нажмите Индекс (). Не все подключаемые модули документации содержат индексы.
